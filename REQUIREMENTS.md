# FluidMusic — 需求总汇 & 开发清单

## 用户核心需求 (逐条记录)

### 一、整体架构
1. 新项目独立目录 `/Users/x/Documents/Codex/FluidMusic`
2. Electron 桌面应用，macOS 优先
3. 窗口固定大小 (默认1200×800)
4. 多层Z轴结构：Layer0流体背景 → Layer1穿透层 → 气泡仓/核心区/控制器/DIY弹窗
5. 穿透层: `pointer-events: none`，只为气泡仓+核心区+控制器启用交互

### 二、流体背景
6. 全屏水波纹实时动态背景
7. 深色基底 (#0a0a14)，多层径向波纹
8. 参考 Mineradio-MacOS ShojiWM liquid-terminal 水波 shader 参数
9. 透明度流体力学感

### 三、中心核心布局 (Center Core)
10. 固定大小，窗口正中偏上
11. **边框: 极窄半透明边框**，不能贴紧内容
12. **边缘四周需要扩大，留有间距** — 热区不能紧贴核心元素
13. 由上到下: 粒子封面 → 歌曲信息 → 泡沫律动
14. 设置按钮在核心区右上角
15. 歌曲名称+作者: **3D环绕效果**，分行显示，歌名大号字体，作者小号

### 四、粒子封面 (Album Art Particles)
16. 参考 Mineradio-MacOS **emily专辑封面 封面粒子·快速入场** 预设
17. 参考 `buildCoverParticleGeometry()` 函数，封装为可复用模块
18. **如沙一般细腻入微** — 粒子颗粒度 1-2px，密度高
19. 网格分辨率: 默认118×118 (可调60~200)
20. 鼠标/触控拖动旋转3D视角
21. 随音频律动: **多层渲染，低/中/高三个频率层分别驱动不同深度粒子**
22. 入场: 粒子散落→聚合 (快速入场, 1.2s)
23. 切歌: 粒子散开→新封面聚合
24. **演示图片: 使用 Mineradio-MacOS build/icon.png**

### 五、泡沫/棉花糖律动 (Foam Element)
25. 参考 Mineradio-MacOS **泡沫 珍珠虹彩·柔波浮沉** 预设
26. 位于歌曲名称下方
27. 多层球体粒子群，**随不同音频频率上下起伏+旋转**
28. **随机变换柔和高级感配色** (珍珠白/虹彩/粉蓝渐变)
29. 半透明+珍珠光泽+柔波浮沉动画

### 六、顶部气泡仓 (账号 + 3D歌单队列)
30. 位于中心核心上方，靠近但**不能离太远**
31. 左侧: 多平台账号登录状态 (**网易云+QQ音乐**，参考Mineradio-MacOS)
32. **支持多端用户同时登录**，提前设计UI布局
33. 右侧: 3D歌曲队列
34. 队列: 歌曲照片横向排列，最中间高亮凸显当前播放 → 两端渐行渐远越小
35. **鼠标移动上去要有生动的迎合动作**

### 七、左侧气泡仓 (歌单列表)
36. 悬浮透明气泡形态
37. 与中心核心保持间距 (≥24px)
38. 居中对称于中心核心
39. **不在中心核心内，悬浮在外部**
40. 触发: 鼠标悬浮中心核心左边缘 → 弹出; 点击左边缘 → 常驻

### 八、右侧气泡仓 (歌词显示)
41. 悬浮透明气泡形态
42. 与中心核心保持间距
43. 居中对称
44. 歌词逐行显示，当前句高亮
45. **上下越远淡化处理** (CSS mask-image gradient)
46. **DIY中设置显示行数**，滑块从0开始，默认全部，0=不显示

### 九、底部音乐控制器
47. 位于中心核心下方，保持距离
48. **取消粒子沙感效果** — 用简洁设计
49. 按钮借鉴 **Mineradio-MacOS 按钮效果**
50. 功能: 上一曲/播放暂停/下一曲/音量/播放模式/进度
51. **频谱: 多层粒子状**，非传统柱状
52. 频谱**3D环绕排列**，鼠标可拖动旋转

### 十、DIY设置弹窗
53. 居中覆盖在核心区域上方 (不在底部)
54. **TAB布局**，分类罗列清楚
55. Tab: 粒子封面/泡沫特效/歌词设置/歌单设置/频谱设置/控制器/背景流体/气泡仓
56. 每个Tab内合理布局配置项

### 十一、配置持久化
57. **上次选择的特效预设 (柱形/泡沫/不规则) 必须记忆**，下次启动恢复
58. 所有DIY设置存入本地持久化存储
59. 歌词显示行数记忆
60. 气泡仓常驻/隐藏状态记忆

### 十二、边缘触发系统
61. 中心核心四周边框外延伸热区 (~30px)
62. 鼠标悬浮边缘 → 对应气泡仓显示 (弹簧动画)
63. 鼠标点击边缘 → 气泡仓常驻/取消常驻
64. **左右气泡仓和中心核心不能贴在一起，要有距离**
65. 气泡仓hover时也保持显示

### 十三、3D歌单队列交互
66. 顶部仓内歌曲队列横向3D排列
67. 中间高亮当前播放，两端渐远渐小
68. 鼠标移动 → 生动的迎合动作 (跟随/弹性)
69. 3D透视旋转

### 十四、音频系统
70. Web Audio API 实时频谱分析 (fftSize 2048)
71. 分离 Bass(20-250Hz) / Mid(250-4kHz) / Treble(4k-20kHz) 三频段
72. 多平台歌单获取 (Node.js Express 后端代理)
73. 网易云+QQ音乐 API 对接

### 十五、动画规范
74. 所有UI状态变化必用弹簧动画，禁止硬切
75. 贝塞尔: 展开 cubic-bezier(0.1,1.1,0.1,1.1) / 收起 cubic-bezier(0.3,-0.3,0,1)
76. 粒子入场: 1.2s ease-out
77. 切歌: 散开0.4s→聚合0.8s
78. 气泡仓: scale(0.88→1.0)+opacity(0→1) 弹簧0.5s
79. 泡沫配色渐变: 每30s随机切换
